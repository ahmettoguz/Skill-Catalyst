########################## NodeJs ###########################
------------------------------------------------------------

Create App:
    Create folder and move into folder.

    With init you can indicate name, description ...
    npm init backend
------------------------------------------------------------

Install Dependencies:
    npm i express
    npm i nodemon
    npm i body-parser
    npm i cors
    npm i bcrypt
    npm i jsonwebtoken
    npm i dotenv
    npm i axios
    npm i mongoose


------------------------------------------------------------

Uninstall Dependencies:
    npm rm express
------------------------------------------------------------

Using local param:
    check route file for how to use location params
------------------------------------------------------------
